Add New Sensor Record

This form will add a new record to the sensors inventory and allow you to assign
which employee is responsible for managing this sensor by using their employee number.

Detailed Instructions:

The sensor number must be four numbers long and begin with "100"

A sensor number of "1004" is pre-populated as an example

There are three different sensor types, "temperature", "humidity" and "pressure"

The sensor status should only hold one of 3 values:

  • 0 = fault
  • 1 = active
  • 2 = offline
